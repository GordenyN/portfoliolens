# PortfolioLens

An AI-powered financial portfolio dashboard that visualizes risk relationships between assets and aggregates semantic news sentiment analysis.

## Features

- **Risk Vibe Map** — Interactive D3 force-directed graph showing portfolio tickers as connected nodes. Node size = portfolio weight, color = sentiment score. Hover edges to see risk relationships, click nodes to open news sidebar.
- **News Dashboard** — Real-time news feed with sentiment scoring per ticker, aggregate statistics (bullish/bearish ratio), and an AI-generated portfolio digest.
- **AI Portfolio Digest** — Automatically generated by Gemini (via n8n workflow) summarizing key news and suggested actions.
- **Vector Search** — Semantic news search powered by Qdrant vector database with Gemini embeddings for deduplication and relevance ranking.
- **Live Rebalancing** — Drag sliders to adjust portfolio weights and see the graph update in real time.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React 19, Vite 8, D3.js |
| Backend | n8n (workflow automation) |
| AI | Gemini (embeddings + text generation), Ollama |
| Vector DB | Qdrant Cloud |
| Storage | Supabase |
| Data Sources | Yahoo Finance RSS, CNBC RSS, CoinDesk RSS |
| Deployment | Vercel (frontend + serverless proxy) |

## Getting Started

### Prerequisites

- Node.js 20+
- npm

### Installation

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
npm run preview
```

## n8n Workflow

The repository includes two n8n workflows:

1. **News Ingestion** — Fetches RSS feeds → AI analysis → embedding → Qdrant upsert → Supabase
2. **Portfolio Query API** — Webhook endpoint that accepts `{tickers: [...]}` → vector search → format → Gemini digest → respond

To use, import the workflow JSON into your n8n instance and activate it.

## Deployment

The project is deployed on Vercel:

```bash
npm i -g vercel
vercel login
vercel --prod
```

The `api/proxy.js` serverless function acts as a CORS proxy to the n8n webhook.
